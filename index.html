bh_workflow by zoudabi

An automation tool which can chain different steps of a task together.

bh_workflow is a Spring-based java project. It can be used to chain different steps of a task together so that the whole job can be done automatically. These steps can be configured by modifying bhprocess.xml under com.binhunt.workflow.conf. The main entrance of the project is com.binhunt.workflow.test.BackendController, and the common interface of different steps is com.binhunt.workflow.itf.Activity.

We built a malware clustering database based on cloud platform. bh_workflow is used to interface the back-end operations with the web portal, for example, preprocessing of jobs, distributing jobs to the private cloud system for handling.

BinHunt: Automatically Finding Semantic Differences in Binary Programs Debin Gao, Michael K. Reiter and Dawn Song In Proceedings of the 10th International Conference on Information and Communications Security (ICICS 2008), pages 238-255, Birmingham, UK, October 2008
